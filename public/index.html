TermiChat is a multiplayer terminal chat client. Download the Linux package, install it to your shell, then connect to the live workspace server with a single command.

Download Linux Package Download install.sh
Fast path

Install

curl -fsSL http://termichat.zeraynce.com/install.sh | sh

Or manual

tar -xzf teamchat-client-linux-amd64.tar.gz
sh install.sh
Run

Open the chat

termichat
  • The installer saves the server and workspace config automatically.
  • First join on a device asks for your handle.
  • Next join on the same device remembers that handle and auto-connects.
  • New joins only see the latest 7 days of logs.
Troubleshooting

If teamchat is not found

export PATH="$HOME/.local/bin:$PATH"

Then run termichat again in the same terminal. New shells will pick it up automatically after install.
