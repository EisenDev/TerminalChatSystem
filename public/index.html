TermiChat is a multiplayer terminal lobby chat. Install it with one curl command, run termichat, then join or create your lobby directly from the terminal app.

Install

Linux installation

curl -fsSL https://termichat.zeraynce.com/install.sh | sh

Windows users (run it on terminal)

powershell -ExecutionPolicy Bypass -Command "irm https://termichat.zeraynce.com/install.ps1 | iex"
Update

Update TermiChat

curl -fsSL https://termichat.zeraynce.com/update.sh | sh
rehash

This updates the existing termichat binary in place, keeps your current config, and refreshes your shell command cache in the current terminal.

Run

Open the chat

termichat
How To Use
  • Enter your lobby name first, then your lobby code.
  • If this device already joined that lobby before, TermiChat auto-joins without asking for your username again.
  • If it is a new lobby or a new device, you will be asked for a username before entering.
  • Type /list in Search Lobby to browse the lobbies already remembered on this device.
Troubleshooting

If termichat is not found in your current shell, run:

export PATH="$HOME/.local/bin:$PATH"

Then run termichat again in the same terminal.
